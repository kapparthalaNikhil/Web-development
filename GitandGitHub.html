Git and Github

    configuration to github using git bash

  1. git --version
    • It verifies the git installation with version number
  2. git config --global user.name "username"
    • this command is used for configuration or connection with github, have to enter username as for the github account
  3. git config --global user.email "email address"
    • this is also configuration for github.so has to enter email address as same of the github account
  4. git config --list
    • this command shows the what creditials you uploaded as username and email_addreses

    Clone and Status

  5. git clone paste_address_of_the_https
    • It cloning/coping a repository on to our locl machine like laptop or PC.
    • You can get/copy the https address by clicking on the code button in current repository on github
  6. git status
    • It displays the state of the code as following types :
    • untracked - new files that git doesn't yet track
    • modified - changed
    • staged - file is ready to be commited
    • unmodified - unchanged

    Add and Commit commad

  7. git add <-file name->
    • adds new or changed file in your working directory to the Git staging area.
    • If we place <-.->point in place of file_name it adds all the files.
    • To add all files at a time we use point symbol like git add .
  8. git commit -m "some message"
    • It is the record of change

    Push Command

  9. git push origin main
    • push/upload the local repository content to remote/github repository
